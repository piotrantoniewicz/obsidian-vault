02Step-by-Step Guide

1

### Create Your Project Folder

On your computer's file system, create a new, empty folder to serve as the home for your project. A suggested name is `Daily Operating System`.

2

### Establish Your 'Brain' File

Inside your project folder, create a markdown file (e.g., `brain.md`) that acts as a personal instruction manual for Cowork. This file should contain your working preferences, team members, communication style, and other standing instructions.

3

### Connect Folder to a Cowork Project

In the Claude Cowork desktop app, create a new project from an existing folder, selecting the folder you just made. Give the project high-level instructions to define its overall purpose and connect the physical folder to the workspace.

```
I'm gonna use this folder and operating system to help me navigate the day-to-day life of a busy professional. You're gonna help me with some superpowers by applying some AI magic to it all.
```
