# I need you to learn Claude Code immediately - here's how.

Allie Miller ogłasza fundamentalny shift w sposobie pracy z AI: twórca Claude Code przez miesiąc nie otworzył IDE, a Claude napisał 497 commitów i 40 000 linii kodu. Dario Amodei potwierdził, że 90% kodu w Anthropic pisze AI. Claude Code to nie tylko narzędzie dla programistów — to "Everything App" dla każdego, od CEO po asystenta, który może budować aplikacje, analizować DNA, zarządzać projektami. Email zawiera konkretny prompt do wywiadu Claude'a z użytkownikiem i budowania spersonalizowanego systemu produktywności.

## Wnioski
- Prompt "TutorAgent + BuilderAgent" (Claude uczy użytkownika po polsku co robi, jednocześnie budując rozwiązanie) to wzorzec idealny do onboardingu nie-technicznego użytkownika w NGO — człowiek uczy się AI przez działanie, a nie przez ćwiczenia.
- "Build Day" (cały dzień hackathonu dla zespołu) zamiast 5-minutowego tutorialu to warunek faktycznego przyswojenia narzędzi — to bezpośredni wzorzec dla warsztatów AI dla NGO: co najmniej 3-5 godzin praktycznego budowania.
- Dane Anthropic: 90% kodu pisze AI, a delegowanie wzrosło z 0% do 20% pracy dla inżynierów w ciągu 6 miesięcy — to twardy argument dla zarządów NGO, że AI nie jest przyszłością, jest teraźniejszością wymagającą natychmiastowej odpowiedzi strategicznej.

## Cytat
> "Increasingly, code is no longer the bottleneck. This conversational, iterative style of building feels fundamentally different from programming. It feels like directing."

## Zastosowanie
Prompt do wywiadu Claude + budowanie spersonalizowanego systemu można zaadaptować jako ćwiczenie onboardingowe na szkoleniach NGO: uczestnik opisuje swój największy problem operacyjny, Claude prowadzi wywiad, a następnie buduje prototypowe rozwiązanie w trakcie warsztatu.
