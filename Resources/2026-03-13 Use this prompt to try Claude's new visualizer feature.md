# Use this prompt to try Claude's new "visualizer" feature

Anthropic wypuścił nową funkcję Claude (beta, dostępna na wszystkich planach): Visualizer — Claude tworzy interaktywne wykresy, diagramy, flowcharty i kalkulatory bezpośrednio w oknie czatu, bez osobnych plików ani paneli. Ryan Carr udostępnia wizard prompt, który przyjmuje dowolną treść (artykuł, newsletter, framework), proponuje kilka wariantów wizualizacji do wyboru i buduje wybraną opcję inline. Gotowy visual można wyeksportować jako samodzielny plik HTML i użyć jako lead magnet, element kursu lub embed na landing page. Najlepiej działa z ustrukturyzowaną treścią: frameworki, procesy krok-po-kroku, porównania.

## Wnioski
- Visualizer działa jak „tablica whiteboard mid-conversation" — zamiast ściany tekstu Claude buduje interaktywny kalkulator lub diagram wprost w chacie, co ułatwia zrozumienie złożonych konceptów.
- Wizard prompt do wizualizacji można użyć na własnej treści kursowej lub artykułach — gotowy output jako HTML nadaje się bezpośrednio jako interaktywny moduł lekcji lub email-gated lead magnet.
- Treści z wyraźną strukturą (listy kroków, porównania, frameworki) dają najlepsze efekty wizualizacji — warto pisać je z myślą o późniejszej wizualizacji.

## Cytat
> „Claude jakby wyciągał tablicę w środku rozmowy, żeby pomóc ci zobaczyć to, co wyjaśnia."

## Zastosowanie
Użyć Visualizer do tworzenia interaktywnych elementów kursów AI dla NGO — np. wizualizacja frameworku oceny gotowości organizacji na AI jako interaktywny diagram w Claude, eksport do HTML i embed w materiałach szkoleniowych.
