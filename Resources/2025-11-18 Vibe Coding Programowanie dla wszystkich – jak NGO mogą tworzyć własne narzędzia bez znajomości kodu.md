# Vibe Coding Programowanie dla wszystkich – jak NGO mogą tworzyć własne narzędzia bez znajomości kodu

Artykuł wprowadza pojęcie vibe codingu jako nowego podejścia do tworzenia oprogramowania, w którym użytkownik opisuje potrzebę w naturalnym języku, a AI generuje działający kod. Autor, [[Bartłomiej Polakowski]] z [[Sektor 3.0]], adresuje tę metodę wprost do NGO borykających się z ograniczonym budżetem na technologie — wskazuje, że zamiast kosztować dziesiątki tysięcy złotych, własne narzędzia cyfrowe można budować za kilkadziesiąt złotych miesięcznie lub za darmo. Artykuł zawiera konkretny przewodnik krok po kroku oraz dwa praktyczne przykłady gotowych narzędzi dla NGO (kalkulator 1% i kalkulator kampanii fundraisingowej).

## Frameworki i metody

**5 kroków vibe codingu dla NGO:**
1. **Zdefiniuj potrzebę** — jasno opisz problem do rozwiązania (np. system rezerwacji, kalkulator darowizn, quiz o misji)
2. **Wybierz narzędzie** — dla NGO bez doświadczenia technicznego: [[Claude]] lub ChatGPT (darmowe konta, bez instalacji); bardziej zaawansowane: Cursor/Windsurf
3. **Opisz wizję** — w naturalnym języku podaj co aplikacja ma robić, jak wyglądać, jakie dane obsługiwać; można dołączyć link do inspiracji
4. **Iteruj i doprecyzuj** — jeśli coś nie działa, opisz problem lub zmianę prostym zdaniem; AI zaproponuje poprawkę
5. **Testuj i wdrażaj** — przetestuj w przeglądarce; zapisz plik HTML lub opublikuj online; AI podpowie też jak dodać wysyłkę mailową

**Porównanie narzędzi do vibe codingu:**
- [[Claude]] (Anthropic) — kompletne aplikacje webowe, analiza danych, automatyzacja; darmowa + ~20 USD/mies.
- ChatGPT ([[OpenAI]]) — tworzenie stron i aplikacji; darmowa + ~20 USD/mies.
- GitHub Copilot — zaawansowane podpowiedzi kodu, wymaga podstaw programowania; ~10 USD/mies.
- Cursor / Windsurf — edytory kodu z AI, przyjazne dla początkujących; darmowe + płatne plany

## Wnioski

- Vibe coding likwiduje barierę kosztową tworzenia własnych narzędzi cyfrowych dla NGO — od miesięcy i dziesiątek tysięcy złotych do godzin i kilkudziesięciu złotych miesięcznie.
- Kluczem do skutecznego vibe codingu jest precyzyjny, ustrukturyzowany opis potrzeby — im dokładniej opisany problem, kontekst i oczekiwany wygląd, tym lepszy efekt.
- Kod wygenerowany przez AI należy do organizacji i jest niezależny od platformy — można go zapisać, archiwizować i kontynuować pracę z innym narzędziem AI.

## Zastosowanie

Artykuł to gotowy materiał do szkolenia z AI dla NGO — można go wykorzystać jako wprowadzenie do modułu o tworzeniu własnych narzędzi bez programowania. Dwa przykłady (kalkulator 1% i kalkulator kampanii) są bezpośrednio użyteczne jako demonstracje dla uczestników kursu Fundraising z AI. Warto też uwzględnić vibe coding jako opcję w rekomendacjach dla NGO, które chcą budować własne rozwiązania bez angażowania zewnętrznych deweloperów.
