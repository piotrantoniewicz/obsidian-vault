# Set up your first Claude Cowork project (fast) 🧙‍♂️

Artykuł prezentuje "wizard prompt" (ponad 900 słów) do automatycznego skonfigurowania projektu Claude Cowork — od struktury folderów, przez pliki skills, po instrukcje projektowe. Prompt prowadzi przez cztery fazy: rozumienie projektu, zasoby i kontekst, powtarzalne workflowy oraz budowanie struktury — całość zajmuje ok. 5 minut. Podejście zakłada, że najlepsze efekty daje podanie konkretnego projektu ("automatyzuję tygodniowe raporty klienta"), nie ogólnej intencji ("chcę spróbować Cowork").

## Frameworki i metody
- **Faza 1 — Rozumienie projektu**: pytania o cel, odbiorcę i powtarzalny proces; model nie pyta ponownie o rzeczy, które może wywnioskować
- **Faza 2 — Zasoby i kontekst**: identyfikacja materiałów referencyjnych (brand guides, szablony, voice docs) i sprawdzenie, czy są gotowe do wgrania
- **Faza 3 — Powtarzalne workflowy**: identyfikacja zadań, które powinny stać się plikami skills — tych wykonywanych regularnie w tej samej formie
- **Faza 4 — Build**: jednorazowe generowanie struktury folderów, plików skills i instrukcji projektowych
- **Iteracja po każdej sesji**: po wykonaniu zadania przez Claude — aktualizacja pliku skill na podstawie feedbacku; każda runda lepsza od poprzedniej

## Wnioski
- Wizard prompt jako narzędzie onboardingu do Cowork skraca czas konfiguracji nowego projektu z godzin do minut — warto mieć go pod ręką przy każdym nowym kliencie lub projekcie.
- Podejście "jeden projekt, potem rozszerzanie" zapobiega rozproszeniu — lepiej dopracować jeden workflow i pozwolić mu "dojrzewać", zanim zreplikuje się wzorzec na kolejne obszary.
- Prawdziwy przyrost jakości następuje przez iteracyjne doskonalenie plików skills — sam setup to tylko punkt startowy, nie gotowy system.

## Zastosowanie
Wizard prompt można bezpośrednio wykorzystać przy onboardingu klientów NGO do Cowork — zamiast ręcznie budować strukturę, można przeprowadzić klienta przez ten proces podczas warsztatu jako interaktywne ćwiczenie. To gotowy materiał dydaktyczny do szkolenia "Jak zbudować swój AI workspace w Claude". Warto przetestować prompt przy konfiguracji kolejnego projektu dla klienta z sektora organizacji społecznych i zapisać wnioski do skills.
