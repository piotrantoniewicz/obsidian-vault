# Everyone should be using Claude Code more

[[Lenny Rachitsky]] argumentuje, że [[Claude Code]] jest najlepszym narzędziem AI dla osób netechnicznych, choć rzadko postrzeganym w ten sposób. Kluczowy insight: Claude Code to nie "narzędzie do kodowania" — to lokalny agent AI działający bezpośrednio na komputerze, zdolny do zarządzania plikami, automatyzacji zadań, analizy danych i interakcji z innymi usługami. Artykuł prezentuje 50 kreatywnych zastosowań zebranych od ponad 500 użytkowników, obejmujących organizację dokumentów, analizę transkryptów spotkań, tworzenie opisów stanowisk, pracę z plikami audio i wiele innych. Centralnym przesłaniem jest zmiana myślenia: zamiast pytać "jak to zakodować?", wystarczy opisać cel.

## Frameworki i metody

- **Instalacja [[Claude Code]]:** Mac: `curl -fsSL https://claude.ai/install.sh | bash`, Windows: `irm https://claude.ai/install.ps1 | iex`, uruchomienie: `claude`
- **5 osobistych use case'ów autora:** czyszczenie pamięci komputera, poprawa jakości screenshotów, pobieranie filmów YouTube, pobieranie obrazów z Google Docs, losowanie zwycięzcy z Google Sheets
- **Model pracy dla osób netechnicznych:** opisz cel w języku naturalnym → Claude Code opracowuje plan → wykonuje zadanie → prezentuje wyniki. Brak znajomości programowania wymagany.

## Wnioski

- [[Claude Code]] jako "lokalny agent AI" ma dostęp do plików, folderów i systemu operacyjnego — to fundamentalna przewaga nad webowymi chatbotami przy zadaniach wymagających pracy na prawdziwych danych.
- Najwyżej cenione zastosowania łączą kilka kroków w jeden workflow — np. pobieranie notatek ze spotkań + identyfikacja wzorców unikania konfliktu przez menedżera (zastosowanie Dana Shippera).
- Osoby netechniczne używają [[Claude Code]] do zadań organizacyjnych (porządkowanie plików, faktury), analitycznych (synteza rozmów z klientami) i kreatywnych (tworzenie treści we własnym głosie).

## Cytat

> Zapomnij, że to się nazywa Claude Code — pomyśl o tym jak o Claude Local albo Claude Agent. To w istocie superinteligentna AI działająca lokalnie, zdolna do działania bezpośrednio na twoim komputerze.
